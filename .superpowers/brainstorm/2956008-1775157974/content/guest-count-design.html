Guest Count Feature — Design Overview

Where it lives, how it looks, how it's edited

Key decisions made for you

  • Per-day, not per-room — stored on DailyPlanRoom (guest count can vary day to day)
  • Default: 1 guest — no extra setup needed for existing plans
  • Range: 1–5 — tap +/− buttons (no free-text entry)
  • Shown on board card — visible at a glance when count > 1
  • Editable in setup + long-press editor — consistent with how type/priority work
